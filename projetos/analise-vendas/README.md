# 📊 Análise de Vendas - Projeto de Data Analytics

Este projeto tem como objetivo realizar uma análise exploratória de dados de vendas, identificando padrões de comportamento, produtos mais vendidos e possíveis oportunidades de melhoria no desempenho comercial.

---

## 🎯 Objetivo

- Explorar e entender os dados de vendas
- Realizar limpeza e tratamento dos dados
- Identificar padrões e tendências
- Gerar insights para apoio à tomada de decisão

---

## 🛠️ Tecnologias Utilizadas

- Python  
- Pandas  
- Matplotlib  
- Jupyter Notebook  

---

## 📂 Fonte dos Dados

Dataset fictício de vendas contendo informações como:

- Produtos  
- Quantidade vendida  
- Preço  
- Data da venda  
- Região  

*(O dataset foi utilizado apenas para fins de estudo.)*

---

## 🔍 Etapas do Projeto

1. **Importação dos dados**
   - Leitura do dataset em formato CSV

2. **Limpeza e tratamento**
   - Remoção de valores nulos
   - Ajuste de tipos de dados
   - Padronização de colunas

3. **Análise exploratória (EDA)**
   - Identificação dos produtos mais vendidos
   - Análise de faturamento por período
   - Comparação de desempenho por região

4. **Visualização de dados**
   - Criação de gráficos para facilitar a interpretação
   - Análise de tendências ao longo do tempo

5. **Geração de insights**
   - Interpretação dos resultados encontrados

---

## 📊 Principais Insights

- Determinados produtos apresentam volume de vendas significativamente maior
- Existe variação no faturamento ao longo do tempo (possível sazonalidade)
- Algumas regiões concentram maior volume de vendas
